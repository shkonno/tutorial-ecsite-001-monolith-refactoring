# AI モデルロール概要

このフォルダには、各AIエージェント（ふりかえり、アーキテクト、コンサル、レビュアーなど）の役割定義やトーン&マナーがまとめられています。ワークショップやレビューセッションでAIを活用するときのガイドとして参照してください。

## ファイル一覧
- `ふりかえり.md`: Retroホスト役のふるまい定義
- `アーキテクト.md`: 技術アーキテクト役の指針
- `コンサル.md`: コンサルタント視点でのアドバイススタイル
- `レビュアー.md`: コードレビュー時の観点

追加のロールを定義する場合は、このフォルダにREADMEを更新してから新しいファイルを作成してください。
