# Apple UI 実装要件メモ

2024年時点の apple.com のUI設計原則をベースに、ECサイト向けに適用するための実装レベルの指針をまとめています。デザインとフロントエンド実装の双方で参照してください。

## グローバルナビゲーション
- 高さ `44px`（デスクトップ時）。背景は `rgba(22,22,23,0.8)`、スクロール時に solid ブラックで固定。
- 要素配置: 左から Apple ロゴ → プライマリカテゴリ（Store / Mac / iPad ...） → 検索アイコン → バッグアイコン。
- ホバー時は背景を `rgba(29,29,31,0.9)` に変化。
- カテゴリラベル: `font-family: "SF Pro", sans-serif`、`font-size: 14px`、`letter-spacing: 0.01em`。ホバー/フォーカスでテキスト色 `#f5f5f7`。
- 検索アイコン押下で `width: 240px` の入力フィールドがフェード表示。背景 `rgba(35,35,37,0.95)`、プレースホルダー `#98989D`。ESCキーまたはオーバーレイ外クリックで閉じる。

## ヒーロー & モジュール構成
- ヒーローセクション: ビューポート高の 75% を基準（`min-height: 600px`）。テキストブロックは左寄せ、`font-size: clamp(44px, 6vw, 64px)`。サブコピーは `font-size: 21px` で `max-width: 520px` に収める。
- 背景ビジュアルにはフルブリード画像を使用し、コントラスト確保のため `linear-gradient` を 8–12% の透明度で重ねる。
- セクション間余白: `padding-block: 120px`。背景色はライト（`#FBFBFD`）とダーク（`#000000`）を交互使用し、ダーク背景時のテキスト色は `#F5F5F7`。
- 2カラムモジュール: `max-width: 980px` で中央配置。カラム比率 `56:44`。モバイル（`≤768px`）は縦積み、上下余白 `32px`。

## マイクロインタラクション
- スクロールで製品サブナビを `top: 0` に固定。出現アニメーションは 200ms のフェード＋スライド（`transition: opacity 200ms ease, transform 200ms ease`）。
- カードホバー時は影を使わず、画像を 1.03 倍にスケール（`transform: scale(1.03)`; `transition: transform 250ms ease-out`）。
- カラーバリエーション: `40px` の円形スウォッチ。ホバー/フォーカスでボーダーを `2px solid #0071E3` に。選択状態はフィル `#0071E3`。
- CTA ボタン（プライマリ）: `background: #0071E3`、ホバーで `#147CE5`、フォーカスリングは `outline: 3px solid rgba(0,113,227,0.35)`。

## 製品詳細ページ（PDP）
- ファーストビュー: 3カラム構成（ギャラリー 45%、詳細コピー 35%、購入モジュール 20%）。モバイル（`≤768px`）では「ギャラリー → コピー → 購入」の順で縦並び。
- 「買う」CTA押下で構成モーダルを表示。モーダル幅 `max-width: 860px`。ステップは「モデル選択 → カラー → ストレージ → 価格」タブで遷移。
- 主要スペックはダーク背景 (`#141414`) のタイムライン表示。各項目カード `padding: 48px 0`、アイコン `64px`、テキスト中央揃え。
- 比較モジュール: カード 3–4件、ラッパー幅 `max-width: 1200px`。カード幅 25%、モバイルでは横スクロール＋スナップポイントを設定。

## タイポグラフィ & カラートークン
- ベースフォント: `SF Pro`, fallback `-apple-system, BlinkMacSystemFont, "Helvetica Neue", sans-serif`。
- ボディコピー: `font-size: 17px`、`line-height: 1.47`、文字色 `#1D1D1F`。
- 強調テキスト: `font-weight: 600`。ラベルや補足文は `font-size: 15px`。小見出しは `font-size: 28px`。
- アクセントカラーは製品カテゴリ単位で `--accent` を切り替え（例: Mac = `#0071E3`, Watch = `#BA0C2F`）。セクションの分割は罫線より余白と背景色差で調整。
- フッター: `background: #1D1D1F`、リンクテキスト `#F5F5F7`。グリッド `grid-template-columns: repeat(4, 1fr)`、文字サイズ `12px`、`padding: 40px 0`。

## 運用上のメモ
- トークン化（色、タイポ、スペーシング）で再利用性を担保。デザインシステムに組み込み、モジュールライブラリ化を想定。
- ブレークポイント: 1280px / 1024px / 768px / 414px を基準にレイアウトとタイポを再確認。
- プロトタイプ作成時は、ヒーロー遷移やサブナビ固定などのモーション仕様を Figma または Framer で共有する。
