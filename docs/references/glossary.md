# 用語集

| 用語 | 説明 |
|------|------|
| App Router | Next.js 16で導入された新しいルーティングシステム。`app/` ディレクトリ配下でページやAPIを定義する。 |
| LocalStack | AWSサービスをローカルで模倣するツール。S3やSecrets Managerなどの検証に使用。 |
| RSC | React Server Components。サーバー側でのレンダリングとデータ取得を組み合わせる仕組み。 |
| ADR | Architecture Decision Record。重要な設計判断を履歴として残す文書。 |
| Fargate | AWSのサーバーレスコンテナ実行基盤。初期構成で採用していた。 |
